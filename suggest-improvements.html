Feature Request Ideas

Not sure what to suggest? Here are some areas where we're looking for input:

New Bookmarklet Ideas

  • Social media tools (download images, clean URLs, etc.)
  • Accessibility helpers (font size, contrast, screen reader support)
  • Developer utilities (performance analysis, SEO checks)
  • Content extraction tools (save articles, export data)
  • Privacy tools (clear tracking, block scripts)

UI/UX Improvements

  • Better mobile experience
  • Search and filtering capabilities
  • Bookmarklet categories and organization
  • Usage statistics and favorites
  • Export/import functionality

Contribution Guidelines

For Bug Reports:

  • Include your browser name and version
  • Describe the expected vs. actual behavior
  • Provide steps to reproduce the issue
  • Include any error messages from the browser console

For Feature Requests:

  • Explain the problem your feature would solve
  • Describe how you envision the feature working
  • Consider the security and privacy implications
  • Think about how it would work across different browsers

For Code Contributions:

  • Follow existing code style and patterns
  • Test your bookmarklets across multiple browsers
  • Consider edge cases and error handling
  • Provide clear documentation and examples
